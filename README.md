# MPro-Town-Planning

## 概要
神の街づくりシミュレーションゲーム

街を発展させつつ住民を増やし、刈るべきその時までじっくり待とう…

## コマンド

### 起動


#### Windows
```powershell
.\gradlew.bat run
```

#### その他OS
```bash
./gradlew run
```

### ビルド

#### Windows
```powershell
.\gradlew.bat build
```

#### その他OS
```bash
./gradlew build
```
